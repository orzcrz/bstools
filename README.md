# mac-config

简化Mac环境的同步，懒人必备

## 使用

**如果是已有环境，请先备份好$HOME下的已有配置文件，脚本涉及部分文件的删除替换。**


```
curl -L https://raw.githubusercontent.com/orzcrz/mac-config/master/setup.sh | bash
```

## 配置项

### 工具包

- homebrew
- oh-my-zsh
- wget
- cookiecutter
- tree
- ruby
- rbenv
- cocoapods
- pyenv

### 软链

- zshrc
- lldbinit
